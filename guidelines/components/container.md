### Container

**Purpose**: Max-width content container for centering page content.

**Import**: `import { Container } from '@anthropic/laurel-design'`

**Props**:
- `size` — Container max-width variant

**Example**:
```tsx
<Container size="lg">
  <Text as="h1" size="3xl">Page Content</Text>
  <Text>Centered within max-width container.</Text>
</Container>
```
