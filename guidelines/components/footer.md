### Footer

**Import**: `import { Footer } from '@anthropic/laurel-design'`

Page footer with link columns.

**Example**:
```tsx
<Footer columns={[{ title: 'Product', links: [{ label: 'Features', href: '/features' }] }, { title: 'Company', links: [{ label: 'About', href: '/about' }] }]} copyright="© 2025 Acme Inc." />
```
