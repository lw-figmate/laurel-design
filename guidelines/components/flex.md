### Flex

**Purpose**: Flexbox layout wrapper for flexible arrangements.

**Import**: `import { Flex } from '@anthropic/laurel-design'`

**Props**: Standard flex properties (direction, align, justify, wrap, gap).

**Example**:
```tsx
<Flex align="center" justify="between" gap="4">
  <Text>Left content</Text>
  <Button>Action</Button>
</Flex>
```
