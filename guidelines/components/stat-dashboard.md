### StatDashboard

**Import**: `import { StatDashboard } from '@anthropic/laurel-design'`

Statistics dashboard with multiple stat cards.

**Example**:
```tsx
<StatDashboard items={[{ label: 'Revenue', value: '$45k', change: '+12%', trend: 'up' }, { label: 'Users', value: '1,234', change: '+5%', trend: 'up' }]} />
```
