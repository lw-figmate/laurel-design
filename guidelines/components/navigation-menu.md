### NavigationMenu

**Import**: `import { NavigationMenu, NavigationMenuItem, NavigationMenuLink } from '@anthropic/laurel-design'`

Top-level navigation bar.

**Example**:
```tsx
<NavigationMenu>
  <NavigationMenuItem>
    <NavigationMenuLink href="/">Home</NavigationMenuLink>
  </NavigationMenuItem>
  <NavigationMenuItem>
    <NavigationMenuLink href="/products">Products</NavigationMenuLink>
  </NavigationMenuItem>
</NavigationMenu>
```
