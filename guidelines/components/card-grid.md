### CardGrid

**Import**: `import { CardGrid } from '@anthropic/laurel-design'`

Responsive grid layout of cards.

**Example**:
```tsx
<CardGrid columns={3}>
  <Card><CardBody>Item 1</CardBody></Card>
  <Card><CardBody>Item 2</CardBody></Card>
  <Card><CardBody>Item 3</CardBody></Card>
</CardGrid>
```
