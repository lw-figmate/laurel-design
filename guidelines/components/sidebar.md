### Sidebar

**Import**: `import { Sidebar, SidebarItem, SidebarSection } from '@anthropic/laurel-design'`

Side navigation panel.

**Example**:
```tsx
<Sidebar>
  <SidebarSection title="Main">
    <SidebarItem href="/dashboard" active>Dashboard</SidebarItem>
    <SidebarItem href="/projects">Projects</SidebarItem>
    <SidebarItem href="/settings">Settings</SidebarItem>
  </SidebarSection>
</Sidebar>
```
