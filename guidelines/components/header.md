### Header

**Import**: `import { Header } from '@anthropic/laurel-design'`

Page header/navigation bar organism.

**Example**:
```tsx
<Header logo={<Text weight="bold" size="lg">Acme</Text>} actions={<Button variant="ghost" size="sm">Sign In</Button>}>
  <Link href="/products">Products</Link>
  <Link href="/about">About</Link>
</Header>
```
