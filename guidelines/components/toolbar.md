### Toolbar

**Import**: `import { Toolbar, ToolbarGroup } from '@anthropic/laurel-design'`

Action toolbar for grouping tools/actions.

**Example**:
```tsx
<Toolbar>
  <ToolbarGroup>
    <Button variant="ghost" size="sm">Bold</Button>
    <Button variant="ghost" size="sm">Italic</Button>
  </ToolbarGroup>
  <ToolbarGroup>
    <Button variant="ghost" size="sm">Align Left</Button>
    <Button variant="ghost" size="sm">Align Center</Button>
  </ToolbarGroup>
</Toolbar>
```
