### DataList

**Import**: `import { DataList, DataListItem, DataListLabel, DataListValue } from '@anthropic/laurel-design'`

Key-value pair data display.

**Example**:
```tsx
<DataList>
  <DataListItem>
    <DataListLabel>Name</DataListLabel>
    <DataListValue>Jane Doe</DataListValue>
  </DataListItem>
  <DataListItem>
    <DataListLabel>Email</DataListLabel>
    <DataListValue>jane@example.com</DataListValue>
  </DataListItem>
</DataList>
```
