### DropdownMenu

**Import**: `import { DropdownMenu, DropdownMenuItem, DropdownMenuSeparator } from '@anthropic/laurel-design'`

**Example**:
```tsx
<DropdownMenu trigger={<Button variant="ghost">Actions</Button>}>
  <DropdownMenuItem onClick={() => {}}>Edit</DropdownMenuItem>
  <DropdownMenuItem onClick={() => {}}>Duplicate</DropdownMenuItem>
  <DropdownMenuSeparator />
  <DropdownMenuItem onClick={() => {}}>Delete</DropdownMenuItem>
</DropdownMenu>
```
