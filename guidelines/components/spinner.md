### Spinner

**Purpose**: Loading spinner indicator.

**Import**: `import { Spinner } from '@anthropic/laurel-design'`

**Props**:
- `size` — `'sm'` | `'md'` | `'lg'` | `'xl'`
- `label` — String (accessible label, default: "Loading")

**Example**:
```tsx
<Spinner size="md" />
<Spinner size="lg" label="Loading data..." />
```
