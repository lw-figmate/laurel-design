### HoverCard

**Import**: `import { HoverCard } from '@anthropic/laurel-design'`

Rich preview content shown on hover.

**Example**:
```tsx
<HoverCard trigger={<Link href="/user/jane">@janedoe</Link>}>
  <Avatar src="/jane.jpg" alt="Jane Doe" size="lg" />
  <Text weight="semibold">Jane Doe</Text>
  <Text size="sm">Product Designer</Text>
</HoverCard>
```
