### Icon

**Purpose**: SVG icon renderer.

**Import**: `import { Icon } from '@anthropic/laurel-design'`

**Props**:
- `children` — ReactNode (SVG paths)
- `size` — `'xs'` (12px) | `'sm'` (16px) | `'md'` (20px) | `'lg'` (24px) | `'xl'` (32px)
- `label` — String (accessible label; omit for decorative icons)

See `overview-icons.md` for available icons and usage patterns.
