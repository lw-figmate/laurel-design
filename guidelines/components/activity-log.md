### ActivityLog

**Import**: `import { ActivityLog } from '@anthropic/laurel-design'`

Event activity feed.

**Example**:
```tsx
<ActivityLog events={[{ id: '1', user: 'Jane', action: 'created project', timestamp: '2h ago' }, { id: '2', user: 'John', action: 'commented', timestamp: '3h ago' }]} />
```
