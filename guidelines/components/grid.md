### Grid

**Purpose**: CSS Grid layout wrapper.

**Import**: `import { Grid } from '@anthropic/laurel-design'`

**Props**:
- `columns` — Number or responsive object
- `spacing` — Gap between items

**Example**:
```tsx
<Grid columns={3} spacing="4">
  <Card>Item 1</Card>
  <Card>Item 2</Card>
  <Card>Item 3</Card>
</Grid>
```
